开始项目

开始之前​

可参考 NetNote 的 开始之前

  • 拥有一个 GitHub 账号，并 配置好 SSH Key
  • 在本地 安装并配置好 Git
  • 在本地 安装并配置好 Node.JS
    • 在本地 安装并配置好 Yarn
  • 推荐使用 Visual Studio Code 代码编辑器
    • 推荐安装 Better Comments 插件

拉取项目​

前往 MathNote Fork 本项目，然后找一个合适的路径，执行以下指令：

# GitHub 克隆仓库（注意替换为自己的用户名：<username>）
git clone git@github.com:<username>/MathNote.git

# 切换目录
cd MathNote

# 安装依赖
yarn

运行项目​

在项目路径下执行：

yarn start

然后通过浏览器访问 http://localhost:3000/

参与项目​

本项目使用了 Docusaurus 构建，使用 Markdown 与 KaTex 编写文档。

新增内容请每章节推送一次，git commit 需规范填写例如（参考历史提交）：

add/update 初中/高中 章节编号 章节标题
add 高中 7.1 平面向量

目前主要参考 人教版 教材，如果有其他版本可以取并集更新上来，目录结构都可按需灵活调整。
