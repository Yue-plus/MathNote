7.3 向量的坐标表示

7.3.1 向量的分解​

平面向量基本定理 如果 e1e_1e1​ 和 e2e_2e2​ 是平面上的两个不平行的向量， 那么对该平面上的任一向量 aaa , 存在唯一的一对实数 a1a_1a1​, a2a_2a2​, 使

a=a1e1+a2e2\Large a = a_1e_1 + a_2e_2a=a1​e1​+a2​e2​

这个定理告诉我们：平面上任一向量都可沿两个不平行的方向分解为唯一一对向量的和.

7.3.2 向量的直角坐标运算​

两个向量的和与差的坐标等于两个向量相应坐标的和与差；数乘向量积的坐标等于数乘上向量相应坐标的积.

一个向量的坐标等于向量终点的坐标减去始点的相应坐标.
