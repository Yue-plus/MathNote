7.4 向量的内积及其运算

7.4.1 向量的内积​

我们把 ∣a∣\lvert a \lvert∣a∣ 与 ∣b∣\lvert b \lvert∣b∣ cos ⟨a,b⟩.\lang a, b \rang.⟨a,b⟩. 的乘积叫做向量 aaa 与 bbb 的 内积. 记作 a⋅b\Large a·ba⋅b , 即

a⋅b=∣a∣∣b∣cos⟨a,b⟩.\Large a·b = \lvert a \lvert \lvert b \lvert cos \lang a, b \rang.a⋅b=∣a∣∣b∣cos⟨a,b⟩.

由上述定义可知，两个向量 aaa 与 bbb 的内积是一个实数，而且它可以是正数、负数或零.
