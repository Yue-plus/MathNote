1.1 集合及其运算

1.1.1 集合的概念​

参考： 集合概念_百度百科

集合概念关键字​

对象：可以感觉到的客观存在以及思想中的事物或抽象符号，都可以称作对象。

集合：把一些能够确定的不同的对象看成一个整体，就说这个整体是由这些对象的全体构成的集合。

元素：集合中每个对象叫做这个集合的元素。

集合有时也简称为 集。 含有有限个元素的集合叫做 有限集，含有无限个元素的集合叫做 无限集。

集合中元素的特性​

  1. 确定性：作为集合的元素，必须是能够确定的。
  2. 互异性：集合中的元素一定是不同的。
  3. 无序性：集合中的元素没有固定的顺序。

集合符号​

∈\Large \in∈ ：属于；

∉\Large \notin∈/ ：不属于；

N\Large \mathbf{N}N ：由全体非负整数构成的集合，叫做 自然数集 或 非负正数集；

N+\Large \mathbf{N} _+N+​ 或 N∗\Large \mathbf{N} ^*N∗ ：非负整数集 内排除 0 的集合，叫做 正整数集；

Z\Large \mathbf{Z}Z ：由全体整数构成的集合，叫做 整数集；

Q\Large \mathbf{Q}Q ：由全体 有理数 构成的集合，叫做 有理数集 ；

R\Large \mathbf{R}R ：由全体 实数 构成的集合，叫做 实数集 ；

1.1.2 集合的表示方法​

列举法​

当集合元素不多时，可以直接将元素列举出来，并写在花括号（ {}\{\}{} ）内，这种表示集合的方法叫做 列举法：

{1,2,3,4,5,6}\Huge \{1, 2, 3, 4, 5, 6\}{1,2,3,4,5,6}

性质描述法​

A={x∈I∣p(x)}\Huge A = \{x∈I | p(x)\}A={x∈I∣p(x)}

在上式中，竖线（ ∣|∣ ）左边表示集合 AAA 中的任意一个元素，并标出元素的取值范围， 竖线（ ∣|∣ ）右边表示只有集合 AAA 内的元素才具有的 特征性质 。

在某种约定下，xxx 的取值合集可省略不写，例如上式可简写为 A=x∣p(x)\Large A = {x | p(x)}A=x∣p(x) 。

1.1.3 集合之间的关系​

集合关系符号​

⊆\Large \subseteq⊆ ：包含于；

⊇\Large \supseteq⊇ ：包含；

∅\Large \varnothing∅ ：空集；

⫋\Large \subsetneqq⫋ ：真包含于；

⫌\Large \supsetneqq⫌ ：真包含；

子集​

如果集合 AAA 的任意一个元素都是集合 BBB 的元素，那么集合 AAA 叫做集合 BBB 的 子集，记作：

A⊆B或A⊇B\Huge A \subseteq B 或 A \supseteq BA⊆B或A⊇B

读作“AAA 包含于 BBB” 或 “BBB 包含 AAA”。

空集​

不含任何元素的集合叫做 空集，记作 ∅\Large \varnothing∅。

空集 是任意一集合的子集。

真子集​

如果集合 AAA 是集合 BBB 的子集，并且 BBB 中至少有一个元素不属于 AAA，那么集合 AAA 叫做 集合 BBB 的 真子集，记作：

A⫋B或B⫌A\Huge A \subsetneqq B 或 B \supsetneqq AA⫋B或B⫌A

读作“AAA 真包含于 BBB”或“BBB 真包含 AAA”。

真子集与子集的区别：

  • 子集就是一个集合中的全部元素是另一个集合中的元素，有可能 与另一个集合相等；
  • 真子集就是一个集合中的元素全部是另一个集合中的元素，但不存在 相等；

集合相等​

如果两个集合的 元素 完全相同，那么就可以说这两个 集合相等。

集合 AAA 等于集合 BBB，记作：

A=B\Huge A = BA=B

由相等的定义，可得：

如果 A⊆BA \subseteq BA⊆B 且 B⊆AB \subseteq AB⊆A，那么 A=BA = BA=B； 反之，如果 A=BA = BA=B，那么 A⊆BA \subseteq BA⊆B 且 B⊆AB \subseteq AB⊆A；

1.1.4 集合的运算​

交集​

给定两个集合 AAA、BBB，由既属于 AAA 又属于 BBB 的所有公共元素所构成的集合，叫做 AAA、BBB 的 交集，记作：

A∩B\Huge A \cap BA∩B

读作“AAA 交 BBB”

并集​

给定两个集合 AAA、BBB，把它们所有的元素合并在一起构成的集合，叫做 AAA 与 BBB 的 并集，记作：

A∪B\Huge A \cup BA∪B

读作“AAA 并 BBB”

全集与补集​

在研究集合与集合之间的关系时，如果一些集合都是某一给定集合的子集，那么称这个给定的集合为这些集合的 全集，通常用 UUU 表示。

如果 AAA 是全集 UUU 的一个 子集，由 UUU 中的所有不属于 AAA 的元素构成的集合，叫做 AAA 在 UUU 中的 补集，记作：

∁UA\Huge \complement _ U A∁U​A

读作“AAA 在 BBB 中的补集”。
