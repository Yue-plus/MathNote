3.2.1 二次、二次问题​

用长为 20m 的绳子围成一个矩形，写出两边长之间的函数关系。 并求出两边长各是多少时，围成的矩形面积最大。

使用图像法​

由上表可看到，矩形的另一边长和矩形面积都是矩形一边长的函数。 用 xxx 表示自变量， yyy 表示对应的函数值。 根据函数表，画出这两个函数的图象：

从函数表和图 fff 中可以看到，矩形的一边越长，另一边越短， 从函数表和图 ggg 中可以看到，当矩形的一边小于 5m 时，函数值随边长增加而增加， 当矩形的一边长为 5m 时，矩形获得最大值。 然后函数值又随着边长的增加而减小。

使用公式法​

设一边长为 xxx ，另一边长为 yyy ，依题意，函数关系可用解析式表述为：

f=10−x(0≤x≤10)\Huge f = 10 - x \quad (0 \le x \le 10)f=10−x(0≤x≤10)

进而，可得矩形面积为：

g=x(10−x)\Huge g = x(10 - x)g=x(10−x)

解二次函数一般使用 配方法（Completing the square） ：

对上面得到的解析式进行如下变形：

y=g(x)=x(10−x)=−x2+10x=−[(x−5)2−25]=−(x−5)2+25\Large \begin{align*} y = g(x) &= x(10 - x) = -x^2 + 10x \\ &= -[(x-5)^2 - 25] \\ &= -(x - 5)^2 + 25 \end{align*}y=g(x)​=x(10−x)=−x2+10x=−[(x−5)2−25]=−(x−5)2+25​

由上式，就可以得到当 x=5\Large x = 5x=5 时，函数取得最大值 25\Large 2525 。

上面变形的方法就是配方法。 在解二次方程时已用到它。

配方法对研究所有的二次函数都适用。 它是解决二次函数问题的通用方法。

3.2.2 一次函数模型​

y=kx+b(b≠0,x∈R)\Huge y = kx + b \quad (b \not = 0, x \in \mathbf{R} )y=kx+b(b=0,x∈R)

上式叫做 一次函数。

已知一次函数的图像为一条直线。 接下来对一次函数的图像再作进一步的分析。

正比例函数​

在上式中，令 b=0b = 0b=0，则函数：

y=kx\Large y = kxy=kx

上式叫做 正比例函数。

先尝试做出函数图像来看看是什么形状：

  1. 在上式中，令 x=0x = 0x=0 ，则 y=0y = 0y=0 。所以上式的图像经过原点 (0,0)(0, 0)(0,0) 。
  2. 设 x1,y1x_1, y_1x1​,y1​ 是上式的任意一组解，作点 A(x1,y1)A(x_1, y_1)A(x1​,y1​) ，因为：
    y1=kx1\Large y_1 = kx_1y1​=kx1​
    所以点 AAA 在正比例函数 y=kxy = kxy=kx 的图像上。
  3. 过点 OOO 和点 AAA 作直线 OAOAOA 。

接下来说明直线 OAOAOA 是一次函数 y=kxy = kxy=kx 的图像：

  1. 设 P(x,y)P(x, y)P(x,y) 为直线 OAOAOA 上任意一点，过点 AAA 和点 PPP 分别作为 XXX 轴的垂线， 垂足为 M,NM, NM,N ，则：
    △OAM∽△OPN\Large \triangle OAM \backsim \triangle OPN△OAM∽△OPN
  2. 由相似三角形的对应边成比例，可推出：
    yx=y1x1=k\Large \frac{y}{x} = \frac{y_1}{x_1} = kxy​=x1​y1​​=k

这说明直线 OAOAOA 上的所有点的坐标都满足函数关系式：

y=kx\Large y = kxy=kx

反之，如果点 PPP 的坐标 (x,y)(x, y)(x,y) 满足 y=kxy = kxy=kx ，则点 PPP 一定在直线 OAOAOA 上。

综合以上两点说明，就可以得到结论： 函数 y=kxy = kxy=kx 的图像是一条过原点和点 AAA 的直线 OAOAOA 。

一般一次函数图像​

从函数 y=kxy = kxy=kx 与 y=kx+by = kx + by=kx+b 可以看出，对于自变量相同的值 y=kx+by = kx + by=kx+b ， 总可以由正比例函数 y=kxy = kxy=kx 的对应值加上 bbb 得到，这表示 y=kx+by = kx + by=kx+b 的图像是由 y=kxy = kxy=kx 的图像沿 yyy 轴方向平移 bbb 个单位得到。

当 x=0x = 0x=0 时 y=by = by=b ，当 y=0y = 0y=0 时 x=−bkx = - \frac{b}{k}x=−kb​ 。

所以一次函数的图像是通过点 (0,b)(0, b)(0,b) 和点 (−bk,0)(- \frac{b}{k}, 0)(−kb​,0) 的一条直线。

一次函数具有下面一些主要性质：

  1. 函数值的改变量与相应自变量的改变量成正比。

    设 (x1,y1),(x2,y2)(x_1, y_1), (x_2, y_2)(x1​,y1​),(x2​,y2​) 是 y=kx+by = kx + by=kx+b 的两组解，则：

    y1=kx1+by2=kx2+b\Large y_1 = kx_1 + b \\ y_2 = kx_2 + by1​=kx1​+by2​=kx2​+b

    整理得：

    y2−y1=k(x2−x1)令Δy=y2−y1Δx=x2−x1则Δy=kΔx\begin{align*} y_2 - y_1 &= k(x_2 - x_1) \\ \text{令} \quad \Delta y &= y_2 - y_1 \\ \Delta x &= x_2 - x_1 \\ \text{则} \quad \Delta y &= k \Delta x \end{align*}y2​−y1​令ΔyΔx则Δy​=k(x2​−x1​)=y2​−y1​=x2​−x1​=kΔx​

  2. 当 k>0k > 0k>0 时，一次函数是增函数； 当 k<0k < 0k<0 时，一次函数是减函数；

3.2.3 二次函数模型​

y=ax2+bx+c(a≠0)\Huge y = ax^2 + bx + c \quad (a \not = 0)y=ax2+bx+c(a=0)

上式叫做 二次函数。 它的 定义域 是 R\mathbf{R}R。

如果 b=c=0b = c = 0b=c=0 ，则上式变为：

y=ax2(a≠0)\Large y = ax^2 \quad (a \not = 0)y=ax2(a=0)

它的图像是一条顶点为原点的抛物线。

当 a>0a > 0a>0 时，抛物线开口向上； 当 a<0a < 0a<0 时，抛物线开口向下；

这个函数为偶函数，yyy 轴为图像的对称轴。

可以看出，函数 y=ax2y = ax^2y=ax2 中的系数 aaa 对函数图形的影响：

  • 当 aaa 从 −3-3−3 逐渐变化到 000 时，抛物线开口向下并逐渐变大；
