8.1 坐标系中的基本公式

8.1.1 数轴上的距离公式和中点公式​

一般地，在数轴上，如果 A(x1),B(x2)A(x_1), B(x_2)A(x1​),B(x2​) , 则这两点之间的距离公式为

∣AB→∣=∣x2−x1∣\Large \lvert \overrightarrow{AB} \lvert = \lvert x_2 - x_1 \lvert∣AB∣=∣x2​−x1​∣

一般地，在数轴上， A(x1),B(x2)A(x_1), B(x_2)A(x1​),B(x2​) 的中点的坐标 xxx 满足关系式

x=x1+x22\Large x = \frac{x_1 + x_2}{2}x=2x1​+x2​​

8.1.2 平面直角坐标系中的距离公式和中点公式​

计算 $A(x_1, y_1) , B(x_2, y_2) 两点的距离公式

∣AB∣=(x2−x1)2+(y2−y1)2\Large \lvert AB \lvert = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}∣AB∣=(x2​−x1​)2+(y2​−y1​)2​

计算数轴上的中点公式，可得

x=x1+x22,y=y1+y22,\Large x = \frac{x_1 + x_2}{2}, y = \frac{y_1 + y_2}{2},x=2x1​+x2​​,y=2y1​+y2​​,

这就是线段中点坐标的计算公式，简称 中点公式.
