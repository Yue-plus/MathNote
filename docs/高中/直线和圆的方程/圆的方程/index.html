8.3 圆的方程

8.3.1 圆的标准方程​

由距离公式得

(x−a)2+(y−b)2\Large \begin{equation} \sqrt{(x-a)^2 + (y-b)^2} \end{equation}(x−a)2+(y−b)2​​​

两边平方， 得

(x−a)2+(y−b)2=r2\Large \begin{equation} (x-a)^2 + (y-b)^2 = r^2 \end{equation}(x−a)2+(y−b)2=r2​​

方程 (2) 就是以 C(a,b)C(a,b)C(a,b) 为圆心，以 rrr 为半径的圆的方程，叫做 圆的标准方程.

如果圆心在原点，这时a=0, b=0, 圆的标准方程就是

x2+y2=r2\Large x^2 + y^2 = r^2x2+y2=r2

8.3.2 圆的一般方程​

任何一个圆的方程都可以表示成

x2+y2+Dx+Ey+F=0\Large \begin{equation} x^2 + y^2 + D_x + E_y + F = 0 \end{equation}x2+y2+Dx​+Ey​+F=0​​

因此，只有当 D2+E2−4F>0D^2 + E^2 - 4F \gt 0D2+E2−4F>0 时， 方程 (3) 才表示一个圆， 这时方程 (3) 叫做 圆的一般方程.
