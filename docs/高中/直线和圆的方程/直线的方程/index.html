8.2 直线的方程

8.2.1 直线与方程​

一般地，在平面直角坐标系中，给定一条直线，如果直线上点的坐标都满足某个方程，而且这个方程的坐标所表示的点都在给定的直线上，那么这个方程叫做 直线的方程.

8.2.2 直线的倾斜角与斜率​

一般地，平面直角坐标系内，直线向上的方向与 xxx 轴正方向所成的最小正角 aaa 叫做这条直线的 倾斜角.

倾斜角不是 90°90 \degree90° 的直线，它的倾斜角 aaa 的正切值叫做这条直线的斜率， 通常用 kkk 表示， 即

k=tana\Large k = tanak=tana

一般地，若 x1≠x2x_1\not =x_2x1​=x2​ , 则过点 P1(x1,y1)P_1(x_1,y_1)P1​(x1​,y1​) 和 P2(x2,y2)P_2(x_2,y_2)P2​(x2​,y2​) 的直线的斜率为

k=y2−y1x2−x1\Large k = \frac{y_2 - y_1}{x_2 - x_1}k=x2​−x1​y2​−y1​​

8.2.3 直线方程的几种形式​

1.直线的点斜式方程和斜截式方程

设直线 lll 上不同于 P0P_0P0​ 的任意一点的坐标为 P(x,y)P(x,y)P(x,y) , 则由 lll 的斜率为 kkk 可知

k=y−y0x−x0\Large k = \frac{y-y_0}{x-x_0}k=x−x0​y−y0​​

整理得

y−y0=k(x−x0)\Large y-y_0 = k(x-x_0)y−y0​=k(x−x0​)

上式叫做直线的 点斜式方程.

一条直线，如果与 yyy 轴交于点 (0,b)(0,b)(0,b), 则称这条直线在 yyy 轴上的 截距.

由直线的点斜式方程可知，如果直线的斜率为 kkk ， 截距为 bbb ， 则直线的方程为 y−b=k(x−0)y-b = k(x-0)y−b=k(x−0), 即

y=kx+b\Large y = kx + by=kx+b

上式叫做直线的 斜截式方程.

2.直线的一般式方程和方向向量

证明二元一次方程

Ax+By+C=0(A2+B2≠0)\Large Ax + By + C = 0(A^2 + B^2 \not =0)Ax+By+C=0(A2+B2=0)

在平面直角坐标系内对应的一定是直线，上式叫做直线的 一般式方程.

如果非零向量 aaa 所在的直线与直线 lll 平行，则称 aaa 为直线 lll 的一个 方向向量.；如果非零向量 nnn 所在的直线与直线 lll 垂直， 则称 aaa 为直线 lll 的一个 法向量.

8.2.4 直线与直线的位置关系​

1.两直线平行或相交

给定平面直角坐标系中的两直线

l1:y=k1x+b1l2:y=k2x+b2\Large l_1:y = k_1x + b_1 \\ l_2:y = k_2x + b_2l1​:y=k1​x+b1​l2​:y=k2​x+b2​
